# 西工大学分积小程序前端代码

本项目为小程序“西工大学分积”的前端代码，使用到的框架有：

 - [WePY：让小程序支持组件化开发的框架](https://wepyjs.github.io/wepy-docs/#/)
 - [kai-ui：支持原生和wepy框架的高颜值组件库](https://github.com/Chaunjie/kai-ui)
